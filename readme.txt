– TitleScene START 버튼 누를 시 PlayScene으로 변경
– TitleScene EXIT 버튼 누를 시 게임 종료
– EndScene 스페이스바 누르면 TitleScene으로 변경
– 플레이어들끼리 겹치지않는 로직 적용
– 적끼리 겹치지않는 로직 적용
– DeltaTime().Render를 사용해 프레임 표기
– Gdiplus라이브러리를 통해 객체마다 안티앨리어싱을 적용해 테두리가 부드럽게 보임
– 마우스 휠 업/다운을 통해 카메라 줌인/줌아웃 가능
– 점보 상태에서는 플레이어 분열/먹이뱉기 불가능 하지만, 장애물과 충돌했을때의 분열은 가능
– 적은 기본적으로 자신으로부터 가장 가까운 먹이로 이동 / 플레이어 탐지 범위 안에 플레이어가 있을때는 적이 플레이어 방향 쪽으로 이동
– 적은 일정한 확률로 시간마다 분열
– 먹이를 뱉을때 속도를 가지고 날아가다가 시간을 지나면 멈춤
– 트랩이 생성되기 전에 이미지를 가져와 애니메이션 구현
– 플레이어 분열시 부스트 속도를 가지고 날아가게 구현

– s: 게임 잠시 멈추고 다시 움직임
– z: 플레이화면에서 z 토글키를 통해 줌인아웃 가능 모드 / 고정 탑뷰 모드 설정가능
– p: 주인공 세포의 크기, 플레이 시간, 먹은 먹이수 출력 On/OFF
– r: 현재 플레이화면 리셋 
– q: 게임 종료 → 최종 플레이 시간, 플레이타임 중 최고 반지름, 플레이타임 동안 먹은 먹이 개수 출력
